What is cv2?
OpenCV (Open Source Computer Vision Library) is a powerful library for:

Image processing

Video processing

Computer vision

Basic machine learning for vision tasks

In Python, OpenCV is accessed using the module name cv2.

What does import cv2 do?
It makes all OpenCV functions available so you can use them in your code, for example:

Read and write images

Detect faces or objects

Apply filters (blur, edge detection, etc.)

Work with videos and webcams

Simple example
import cv2

img = cv2.imread("image.jpg")   # Read an image
cv2.imshow("My Image", img)     # Show the image
cv2.waitKey(0)                  # Wait for a key press
cv2.destroyAllWindows()         # Close the window
Common uses of cv2
Image resizing, cropping, rotating

Edge detection (Canny)

Face detection

Video capture from a camera

Drawing shapes and text on images
